# Product Vision

Vancoillie Mailbox aims to become a premium central mailbox for multiple email providers.

The product should feel faster, calmer, and more intentional than Gmail, Outlook, or Apple Mail while preserving professional mailbox power.

## Design Standard

The interface should feel like Apple, Linear, Arc Browser, and Notion designed a mailbox together:

- minimal
- premium
- spacious
- typographically precise
- calm by default
- powerful through context
- responsive from the first screen

## Product Promise

Users should feel that email became simpler without becoming weaker.
